# Narcisse-Konan-Portfolio
Site Web conçu et personnalisé pour présenter mon profil, mes compétences et mes projets. Developpé en HTML, CSS et JavaScript, avec une architecture multi-fichiers. Fonctionnalités : animation au scroll, curseur personnalisé, barres de compétences dynamiques, formulaire de contact intégré via EmailJS et design dark moder responsive.
